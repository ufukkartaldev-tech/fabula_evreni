### Technical Stack
- **Frontend**: Next.js 16, React 19, TypeScript 5
- **Styling**: Tailwind CSS 4
- **Backend**: Firebase (Firestore, Auth, Storage)
- **Testing**: Vitest, Playwright, Testing Library
- **CI/CD**: GitHub Actions
- **Hosting**: Vercel

### Features

#### Story System
- Create traditional linear stories
- Create interactive branching stories
- Community mode - users can contribute branches
- Solo mode - author controls all branches
- Chain mode - collaborative storytelling
- Rich text editor with React Quill
- Draft auto-save system
- Story preview before publishing
- Category system
- Tag system

#### User System
- User authentication (Email, Google)
- User profiles with avatars
- XP and leveling system
- 10-tier badge system
- User statistics
- Profile customization

#### Social Features
- Follow/unfollow users
- Like stories and comments
- Comment on stories
- Reply to comments
- Share stories on social media
- Real-time notifications
- User feed with followed authors

#### Gamification
- XP for reading and writing
- Level progression (1-10)
- Badge collection
- Leaderboard
- Achievements

#### Search & Discovery
- Search stories by title, content, author
- Filter by category
- Filter by type (linear/interactive)
- Sort by date, popularity, likes
- Trending stories
- Recommended stories
